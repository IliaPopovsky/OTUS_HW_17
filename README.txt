Игра Тетрис для ОС Windows. Для вращения фигур нужно нажимать клавишу space на клавиатуре, для ускорения спуска фигур клавишу стрелка вниз на клавиатуре.

Tetris game for Windows OS. To rotate the figures, you need to press the space key on the keyboard, to speed up the descent of the figures, the down arrow key on the keyboard.
